# 概念

Synchronized 是通过对象内部的一个叫做*监视器锁（monitor）* 来实现的。
1. 监视器锁是依赖于底层操作系统的 **Mutex Lock** 来实现的。
2. 操作系统实现线程之间的切换，需要*在用户态和核心态之间进行转换*，成本非常高，状态之间的转换需要相对比较长的时间；
3. 这也 Synchronized 效率低的原因。

因此，这种**依赖于操作系统Mutex Lock所实现的锁我们称之为“重量级锁”**。
